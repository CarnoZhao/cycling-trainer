# 功率区间与热身冷身规则

> 被 commands/plan_generation.md、workout_generation.md 等引用

## 基于FTP的功率区间计算

| 区间 | 名称 | 功率范围 | 用途 |
|------|------|----------|------|
| Z1 | 恢复区 | < 55% FTP | 恢复日、热身初期 |
| Z2 | 有氧区 | 56-75% FTP | 基础有氧、LSD |
| Z3 | 节奏区 | 76-90% FTP | 节奏骑、长距离 |
| Z4 | 阈值区 | 91-105% FTP | Threshold训练 |
| Z5 | VO2max | 106-120% FTP | 高强度间歇 |
| Z6 | 无氧区 | 121-150% FTP | 冲刺训练 |
| Z7 | 神经区 | > 150% FTP | 爆发力训练 |

## 热身冷身功率限制

基于当前 FTP {{FTP}}w：

- **热身**: ramp **48%-71% FTP** (如 125w-185w for FTP 260)
- **冷身**: ramp **71%-48% FTP** (如 185w-125w for FTP 260)
- **组间恢复**: **58% FTP 左右** (如 150w for FTP 260)

具体数值根据 FTP 按比例计算，确保热身冷身在 Z1-Z2 区间。
