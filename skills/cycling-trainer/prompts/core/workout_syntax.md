# Workout 描述语法规则

> 被 commands/workout_generation.md、workout_adjustment.md 等引用

## 基本格式

```
- {时长}m @{功率}w          # 固定功率
- {时长}m ramp {开始}w-{结束}w   # 渐进式ramp

{组数}x                     # 重复组标记（可选）
- {时长}m @{功率}w          # 组内步骤
- {时长}m @{功率}w          # 恢复/间歇
```

## 语法规则

1. **时间单位**: `m` = 分钟，`s` = 秒
2. **功率表示**: 使用绝对功率值 `@ 250w`，不使用百分比
3. **Ramp**: `ramp {开始}w-{结束}w` 表示渐进式增减
4. **重复组**: 数字 + `x` 开头，后面跟缩进的步骤
5. **结构**: 热身 → 主课 → 冷身

## 示例

### 简单阈值课
```
-10m ramp 125w-185w
-20m @ 260w
-10m ramp 185w-125w
```

### 间歇课（VO2max）
```
-10m ramp 125w-185w
5x
-4m @ 310w
-4m @ 150w
-10m ramp 185w-125w
```

### Z2 恢复骑
```
-10m ramp 125w-175w
4x
-5m 165w
-5m 175w
-10m ramp 175w-125w
```

## 强度课命名规则

格式: `{类型} {组数}x{时间}@{功率}w`

- `Threshold 2x20min@260w`
- `VO2max 5x4min@310w`
- `SS 4x10min@220w`

**必须使用绝对功率值**，不要用百分比（如 `@100% FTP` ❌）
