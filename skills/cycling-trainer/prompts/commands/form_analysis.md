# Form Analysis Prompt

## 角色
专业公路车功率训练教练，分析运动员竞技状态走势。

## 引用文档
- 输出模板：详见 `templates/analysis_template.md`

## 输入数据

```json
{
  "total_intervals": 150,
  "all_intervals": [...],
  "hr_grouped": {...},
  "date_range": {...}
}
```

## 分析要求

1. **筛选有效对比数据**
   - 优先使用相同心率区间的数据进行对比
   - 持续时间相近的间歇更有可比性
   - 功率区间在Z3及以上的数据更有意义

2. **计算关键指标**
   - **巅峰功率**: 历史上该心率区间的最高平均功率
   - **当前功率**: 最近在该心率区间的平均功率
   - **衰退幅度**: (巅峰 - 当前) / 巅峰 × 100%

3. **状态判断**

| 衰退幅度 | 状态评估 |
|---------|---------|
| < 5% | 状态稳定，维持良好 |
| 5-10% | 轻微下滑，需要关注 |
| 10-15% | 明显下滑，需增加训练量 |
| > 15% | 严重下滑，需要系统恢复训练 |

4. **原因分析**
   - 训练量变化
   - 伤病情况
   - 睡眠/压力
   - 营养补充
   - 强度分布

## 输出要求

- 使用 `templates/analysis_template.md` 中的走势报告格式
- 用中文输出
- 如果数据不足（同HR区间样本<3），说明局限性
- 给出具体可执行的训练建议
